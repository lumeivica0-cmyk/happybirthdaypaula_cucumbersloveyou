# Тумани Волмур-Голлоу (Mists of Wolmoor Hollow)

Містична детективна 2D-пригода. Подарунок на День народження для Паули 🎁

Стек: **React + TypeScript + Vite**. Увесь візуал (персонажі, локації) намальовано
процедурно на `<canvas>` — жодних зовнішніх image-файлів не потрібно, тож проєкт
збирається "з коробки".

## Запуск локально

```bash
npm install
npm run dev
```

Відкриється на `http://localhost:5173`.

## Збірка для продакшену

```bash
npm run build
npm run preview   # локально перевірити білд
```

## Деплой на Vercel

1. Заведи новий проєкт на [vercel.com](https://vercel.com), імпортуй цей репозиторій
   (або перетягни папку через Vercel CLI: `npx vercel`).
2. Framework Preset — **Vite** (визначиться автоматично).
3. Build Command: `npm run build`, Output Directory: `dist` (стандартно для Vite).
4. Деплой — і посилання готове дарувати.

## Керування

- **WASD** або **стрілки** — рух
- **E** — взаємодія з підсвіченим об'єктом
- Кнопки **Журнал** / **Інвентар** у верхньому правому куті

## Структура проєкту

```
src/
  types.ts             # спільні типи
  data/
    content.ts          # тексти доказів, предметів, діалогів
    locations.ts        # карта локацій, перешкоди, hotspots
  engine/
    sprites.ts           # процедурний рендер піксель-арт персонажів
    useGameState.ts       # стан гри + автозбереження (localStorage)
  components/
    GameCanvas.tsx        # рух, колізії, рендер сцени
    HUD.tsx, Journal.tsx, Inventory.tsx, DialogueBox.tsx, ...
    puzzles/              # три головоломки гри
```

## Нотатка про спрайти

Персонажів намальовано кодом (шари прямокутників на canvas) відповідно до опису
зовнішності — це свідомий вибір, щоб гра працювала повністю без зовнішніх
image-асетів і без жодних реальних фотографій. Якщо захочеш замінити на власний
pixel-art spritesheet (PNG), достатньо переписати `drawCharacter` у
`src/engine/sprites.ts` на завантаження текстури й малювання кадру з неї —
решта коду (анімації, напрямки) підлаштовувати не потрібно, інтерфейс функції
можна залишити тим самим.

## Головоломки (без спойлерів для гравця — тільки для розробника)

1. **Бібліотека** — порядок символів зчитується з нотатки про свічки (кабінет).
2. **Годинникар** — час 8:15 зчитується зі сторінки щоденника (майстерня).
3. **Скриня** — код `683`: 6 (свічок) · 8 (година) · 3 (ворони на стіні),
   збирається з трьох доказів по всій грі.
